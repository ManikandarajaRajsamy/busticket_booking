# Angular CLI configuration file

The `angular.json` file is the Angular CLI configuration file for this project.

## Key sections:
- `projects`: Defines each project in the workspace
- `architect`: Build configurations including:
  - `build`: Production build options
  - `serve`: Development server configuration
  - `e2e`: End-to-end testing configuration

## Common Commands

- `ng serve`: Start development server at http://localhost:4200
- `ng build`: Create production build
- `ng test`: Run unit tests
- `ng lint`: Run linter
